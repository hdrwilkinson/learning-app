## Part 1: Course Modes

Course Modes are the core learning loop. Each mode is directly tied to **Information Points** in your courses:

| Mode | Purpose | When to Use |
|------|---------|-------------|
| **Learn** | Introduction to new concepts | Meeting an IP for the first time |
| **Quiz** | Active recall testing | Daily review, reinforcement |
| **Reflect** | Understanding mistakes | After wrong answers in Quiz |

### The Science Behind Course Modes

Each mode is designed around specific [learning science principles](/docs/concepts/learning-science):

| Mode | Primary Principles | How They're Applied |
|------|-------------------|---------------------|
| **Learn** | **Elaboration** | Connecting new concepts to prior knowledge, explaining in your own words |
| **Quiz** | **Retrieval Practice**, **Spaced Repetition** | Active recall (not re-reading), optimally timed review |
| **Reflect** | **Elaboration** | Deep processing of mistakes, understanding *why* you were wrong |

**Interleaving** appears across modesâ€”your daily queue mixes items from different lessons, and question types vary within Quiz Mode.

## Learn Mode

**Purpose:** Introduce new Information Points with AI-guided explanation.

> **Primary Principle: Elaboration**
> 
> Learn Mode creates rich memory traces by connecting new information to what you already know, encouraging you to ask "why?" and explain concepts in your own words.

### What Happens

1. AI presents an IP's content (explanation, examples)
2. You can ask questions, request examples, or say "I got it"
3. Periodically, AI asks verification questions to confirm understanding â† *Retrieval Practice*
4. Once verified, you move to the next IP

## Quiz Mode

**Purpose:** Test recall through varied question types.

> **Primary Principles: Retrieval Practice + Spaced Repetition**
> 
> Quiz Mode is the core learning mechanic. You're not re-readingâ€”you're actively retrieving information from memory, which strengthens learning far more than passive review. Questions appear at optimally spaced intervals.

### What Happens

1. System selects IPs that are due for review â† *Spaced Repetition: optimal timing*
2. Questions are presented one at a time
3. You answer; system evaluates â† *Retrieval Practice: active recall*
4. Correct â†’ mastery increases, stability grows
5. Wrong â†’ mastery decreases, option to reflect

### Question Types

| Type | Description | Example |
|------|-------------|---------|
| **Binary (T/F)** | True or false statement | "Variables in Python are statically typed. T/F?" |
| **Multiple Choice** | 4 options, 1 correct | "What does `len()` return for a string?" |
| **Question & Answer** | Free-form response | "Explain the difference between `=` and `==`." |

Question type is selected based on your current mastery:
- Low mastery â†’ easier types (binary, multiple choice)
- High mastery â†’ harder types (Q&A, comparisons) â† *Desirable Difficulty*

## Reflect Mode

**Purpose:** Help you understand concepts you got wrong.

> **Primary Principle: Elaboration**
> 
> When you get something wrong, shallow correction ("the answer was B") doesn't help. Reflect Mode uses deep processingâ€”understanding *why* you were wrong, connecting your misconception to the correct concept, and generating new understanding.

### What Happens

1. Triggered after a wrong answer (or before answering if you choose)
2. AI explains the concept from a different angle â† *Elaboration: multiple perspectives*
3. Identifies where understanding broke down â† *Elaboration: connecting gaps*
4. Verifies correction before returning to quiz â† *Retrieval Practice*

## Part 2: Explore (Curiosity Zone)

**Explore is a separate feature from Course Modes.**

While Course Modes are structured around Information Points, Explore is free-form conversationâ€”like ChatGPT. You can ask anything, explore any topic, and follow your curiosity wherever it leads.

> **Primary Principle: Elaboration**
> 
> Explore is elaboration without constraints. You ask questions, explore connections, and generate understanding by following your interests. This creates rich, interconnected knowledge that transfers to new situations.

### What Happens

1. You can ask anythingâ€”no constraints
2. AI responds with rich information
3. Optionally connects to course content if relevant
4. Discoveries can be saved as new course content

### Two Contexts

Explore comes in two flavors:

#### In-Course Explore

- Within a course: "Explore" tab
- Anytime you want to learn something new

### What It Does to Your Data
- Conversations are saved for reference
- Can trigger course creation (from interesting discoveries)
- Can add IPs to existing courses (via "Save to Course")

### Key Behaviors
- No constraints on topics
- Encourages tangents and exploration
- Offers to save/structure valuable discoveries
- **Not tied to specific IPs**â€”this is the key difference from Course Modes
